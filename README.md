# ChatApp 💬

## 📌 Overview

ChatApp is a Java-based messaging application that allows users to register, log in, and exchange messages. The application demonstrates core programming concepts such as object-oriented design, user authentication, message management, and data storage.

---

## ⚙️ Features

* User registration and login validation
* Username and password verification
* Message creation and management
* Message storage using JSON
* Unit testing for application functionality
* Interactive console-based user interface

---

## 🧠 Technologies Used

* Java
* Maven
* JSON Data Storage
* JUnit Testing Framework
* NetBeans IDE

---

## ▶️ How to Run

1. Open the project in NetBeans or your preferred Java IDE.
2. Ensure all Maven dependencies are installed.
3. Build the project.
4. Run the `ChatApp.java` file to start the application.

---

## 📂 Project Structure

* `ChatApp.java` – Main application entry point
* `Login.java` – User registration and login functionality
* `Message.java` – Message creation, validation, and storage
* `messages.json` – Stores message data
* `MessageTest.java` – Unit tests for message-related features
* `pom.xml` – Maven configuration and dependencies

---

## 🔐 Key Functionalities

### User Authentication

* Register new users

* Secure login verification

### Messaging System
